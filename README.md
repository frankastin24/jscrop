# JS Crop - Javascript image cropper with file upload

Usage:

`const options = { 
image: "image_url",
add_styles: true,
upload: true,
upload_url: "upload_url",
upload_fields :
[
['field_1_key' , 'field_1_value'],
['field_2_key' , 'field_2_value']
],
filename : 'filename'
};
JSCrop.create(options);`
